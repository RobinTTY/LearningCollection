Sum Open on JetBrains Academy

You have the file sums.txt. It has multiple lines that contain two numbers separated by whitespace. All numbers are positive.

Write a program that reads this file and print the sum of numbers on each line. So, if the file has n lines, you should print n sums.

Since lines are processed as strings, use split() to get two numbers. Just don't forget to convert them to int.



Memory limit: 256 MB
Time limit: 15 seconds
